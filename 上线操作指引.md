# 首次上线操作指引(只做一次)

全程在浏览器里点点点,不需要命令行。分三大步:**A 上传到 GitHub → B 用 Netlify 发布 → C 打通后台登录**。
遇到任何一步卡住,把截图发给维护者即可。

> 说明:凡是注册、登录、输入密码的步骤,请你自己操作,不要把密码告诉任何人(包括 AI)。

---

## A. 把网站放到 GitHub(你的云端仓库)

1. 打开 https://github.com ,注册并登录(有账号就直接登录)。
2. 右上角 **+** → **New repository**。
   - Repository name:随便起,例如 `research-notes`
   - 选 **Public**(公开)或 **Private**(私有)都行——网站本身都能公开访问;私有只是别人看不到源代码。
   - 其它默认,点 **Create repository**。
3. 进入空仓库页面,点 **“uploading an existing file”**(或 Add file → Upload files)。
4. 打开电脑上的 `website` 文件夹,**把里面的所有内容**(index.html、admin、assets、data、reports、trades、scripts、netlify.toml 等)**全选,拖进网页上传区**。
   - ⚠️ 注意:上传的是 `website` 文件夹 **里面的东西**,让 `index.html` 处在仓库最外层。
5. 下方点 **Commit changes**。上传完成。

---

## B. 用 Netlify 发布(拿到网址、以后自动更新)

1. 打开 https://www.netlify.com ,点 **Sign up**,选择 **“Sign up with GitHub”**(用 GitHub 一键登录最省事)。
2. 登录后点 **Add new site → Import an existing project**。
3. 选 **GitHub**,授权,然后选中你刚建的仓库 `research-notes`。
4. 发布设置(通常会自动读好,确认一下即可):
   - **Build command**:`node scripts/build-index.mjs`
   - **Publish directory**:`.`
5. 点 **Deploy**。等一两分钟,会得到一个网址,像 `random-name-123.netlify.app`。
6. (可选)点 **Site configuration → Change site name**,把网址前缀改成你喜欢的名字。

👉 现在访问那个网址,你的网站就已经公开在线了!

---

## C. 打通后台登录(让你能进 /admin 发布)

这一步稍微多几下,做完就一劳永逸。目的是让后台能用你的 GitHub 账号登录。

### C1. 在 GitHub 建一个“登录凭证(OAuth App)”

1. GitHub 右上角头像 → **Settings** → 最左下 **Developer settings** → **OAuth Apps** → **New OAuth App**。
2. 填写:
   - **Application name**:随便,例如 `research-notes-admin`
   - **Homepage URL**:你的 Netlify 网址,例如 `https://你的名字.netlify.app`
   - **Authorization callback URL**:填 `https://api.netlify.com/auth/done`(照抄)
3. 点 **Register application**。
4. 记下 **Client ID**;再点 **Generate a new client secret**,记下那串 **Client Secret**(只显示一次)。

### C2. 把凭证告诉 Netlify

1. 回到 Netlify,进你的站点 → **Site configuration** → **Access & security** →(向下找)**OAuth** → **Install provider**。
2. 选 **GitHub**,把上一步的 **Client ID** 和 **Client Secret** 粘进去,保存。

### C3. 填一下仓库名

1. 回到 GitHub 你的仓库,打开 `admin/config.yml` 文件,点铅笔图标编辑。
2. 把这一行:
   ```
   repo: your-username/your-repo
   ```
   改成你自己的,例如:
   ```
   repo: 你的GitHub用户名/research-notes
   ```
3. 拉到底点 **Commit changes**。

### C4. 登录后台

1. 打开 `https://你的网址/admin`。
2. 点 **Login with GitHub**,授权。
3. 进去后就能看到“报告 / 交易日志 / 价格”三个栏目,开始写吧!

---

## 完成后,你的日常就只剩:

**打开 `你的网址/admin` → 打字 / 改价格 → 点发布 → 几分钟后网站自动更新。**

详细日常用法见《README-中文使用说明.md》。
